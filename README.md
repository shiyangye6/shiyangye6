### 我相信的几件事

**真实比完美更重要。** 我不假装已经成功了。迷茫、试错、失败、复盘和一点点进步，都是这个主页的一部分。

**AI 不是魔法，是杠杆。** 真正有价值的不是“用了某个工具”，而是把问题拆清楚，把流程跑通，把结果检查明白。

**打工是起点，不是终点。** 我不鼓吹裸辞，也不贩卖焦虑。稳定工作可以是启动资金，但不能是一生唯一的可能性。

**普通人变强，不靠逆袭，靠系统。** 每天做一点正确的小事，让时间成为盟友。
